CHEF_MAAHI Restaurant Website 🍽️
A responsive, single-page restaurant landing website built with HTML & CSS (Bootstrap-powered). It showcases the Chef Maahi brand with a hero banner, menu highlights, delivery/payment info, and social links — a great starting template for a restaurant or food-delivery business site.

✨ Features
Hero Banner — eye-catching landing section with call-to-action buttons
Why Choose Us — highlights on food service, fresh ingredients, and special offers
Explore Menu — categorized menu cards (Non-Veg Starters, Veg Starters, Soups, Fish & Seafood, Main Course, Noodles, Salads, Desserts)
Healthy & Organic Food section with supporting imagery
Delivery & Payment section showing supported payment methods (Visa, Mastercard, PayPal, American Express)
Customer Thank-You / Gift Voucher modal popup
Follow Us section linking to Instagram, Twitter/X, and Facebook
Fully responsive layout using Bootstrap's grid system
🛠️ Built With
HTML5
CSS3 (custom style.css)
Bootstrap 5.3.8 & Bootstrap 4.5.2 (CDN)
Bootstrap Icons
Font Awesome (via Kit CDN)
jQuery & Popper.js (for Bootstrap components)

🚀 Getting Started
No build tools or dependencies to install — it's a static site.

Clone the repository
git clone https://github.com/chefmaahi/CHEF_MAAHI-RESTAURANT-WEBSITE.git
Navigate into the project folder
cd CHEF_MAAHI-RESTAURANT-WEBSITE
Open index.html in your browser — that's it!
Or serve it locally, e.g. with VS Code's Live Server extension, or:
python -m http.server
